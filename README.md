# Multi-Screen Stock Chart

東証プライム市場全銘柄対応の株価チャート分析ツール

## ダウンロード

最新版: [Releases](https://github.com/jimmy0102/multi_screen_stock_chart/releases/latest)

## 基本的な使い方

### 画面構成
3つのチャート画面が同期して表示されます（日足・週足・月足）

### キーボードショートカット

#### 銘柄の切り替え
- **↑ 矢印キー**: 次の銘柄表示
- **↓ 矢印キー**: 前の銘柄表示

#### ウォッチリストフィルタ
- **→ 矢印キー**: すべて → 銅 → 銀 → 金 → ...
- **← 矢印キー**: すべて → 金 → 銀 → 銅 → ...

#### ウォッチリスト登録
- **Enter キー**: 銅 → 銀 → 金 → なし → ... （順番に切り替え）
- **1 キー**: 銅に登録
- **2 キー**: 銀に登録
- **3 キー**: 金に登録
- **0 キー**: ウォッチリスト解除

#### 水平線の操作
- **h キー + クリック**: クリック位置に水平線を描画
- **水平線上で右クリック**: 水平線を削除

### マウス操作
- **拡大/縮小**: マウスホイール
- **スクロール**: ドラッグ
- **価格確認**: チャート上をマウスオーバー

### データ更新
毎朝6時（JST）に自動的に最新データに更新されます。

## インストール方法

### Windows
1. [Releases](https://github.com/jimmy0102/multi_screen_stock_chart/releases/latest)から`.exe`ファイルをダウンロード
2. 警告が出たら「保持」をクリック
3. ダブルクリックして実行
4. 「WindowsによってPCが保護されました」と表示されたら:
   - 「詳細情報」をクリック
   - 「実行」をクリック

### macOS
1. [Releases](https://github.com/jimmy0102/multi_screen_stock_chart/releases/latest)から`.dmg`ファイルをダウンロード
2. DMGファイルをダブルクリック
3. アプリケーションをApplicationsフォルダにドラッグ

## システム要件

- Windows 10/11（64-bit）
- macOS 10.12以降
- 512MB以上の使用可能メモリ
