## 📥 インストール方法

### macOS

#### ステップ1: ダウンロード

[Releases](https://github.com/jimmy0102/multi_screen_stock_chart/releases/latest)から
お使いのMacに合ったファイルをダウンロード:
- **Apple Silicon（M1/M2/M3）**: `MultiScreenStockChart-1.1.0-mac-arm64.dmg`
- **Intel Mac**: `MultiScreenStockChart-1.1.0-mac-x64.dmg`

#### ステップ2: インストール

1. ダウンロードした `.dmg` ファイルをダブルクリック
2. アプリをアプリケーションフォルダにドラッグ&ドロップ

#### ステップ3: 初回起動（⚠️ 重要）

**macOSの最新セキュリティにより、初回起動時に「検証できませんでした」というエラーが表示されます。**

##### 解決方法:

1. アプリをダブルクリックして起動を試みる（エラーが出る）

2. **システム設定** を開く
   - Dock の「システム設定」アイコンをクリック
   - または 画面左上のAppleメニュー → 「システム設定」

3. **「プライバシーとセキュリティ」** をクリック

4. 下にスクロールすると、以下のメッセージが表示されています:
   ```
   "Multi-Screen Stock Chart"は開発元を確認できないため、
   使用がブロックされました。
   ```

5. **「このまま開く」** ボタンをクリック

6. Macのパスワードを入力

7. 確認ダイアログで **「開く」** をクリック

8. アプリが起動します ✅

**💡 ヒント:**
- この操作は**初回のみ**必要です
- 一度許可すれば、次回からは普通に起動できます
- 右クリック→「開く」でも起動できますが、最新のmacOSでは上記の方法が確実です

##### なぜこのエラーが出るのか？

Apple Developer署名されていないアプリは、macOSのセキュリティ機能（Gatekeeper）によりブロックされます。
個人開発のため署名していませんが、ソースコードは全て公開されており、安全なアプリです。

##### 代替方法（ターミナルを使う場合）

ターミナルに慣れている方は、以下のコマンドでも解決できます:

```bash
xattr -cr /Applications/Multi-Screen\ Stock\ Chart.app
```

実行後、アプリを起動してください。

### Windows

#### ステップ1: ダウンロード

[Releases](https://github.com/jimmy0102/multi_screen_stock_chart/releases/latest)から
**ポータブル版（推奨）** をダウンロード:
- `MultiScreenStockChart-1.1.0-win-x64-portable.exe`

#### ステップ2: 実行

1. ダウンロードしたファイルをダブルクリック

2. ダウンロード時に警告が出たら **「保持」** をクリック

3. 実行時に **「WindowsによってPCが保護されました」** と表示されたら:
   - **「詳細情報」** をクリック
   - **「実行」** をクリック

4. アプリが起動します ✅

**💡 ヒント:**
- ポータブル版はインストール不要で、すぐに使えます
- デスクトップにショートカットを作りたい場合は、ファイルを右クリック → 「ショートカットの作成」

#### Windows警告について

開発者署名がないため警告が表示されますが、ウイルスではありません。
ソースコードは全て公開されており、安全なアプリです。

## 📖 基本的な使い方

### 画面構成
3つのチャート画面が同期して表示されます（日足・週足・月足）

### キーボードショートカット

#### 銘柄の切り替え
- **↑ 矢印キー**: 次の銘柄表示
- **↓ 矢印キー**: 前の銘柄表示

#### ウォッチリストフィルタ
- **→ 矢印キー**: すべて → 銅 → 銀 → 金 → ...
- **← 矢印キー**: すべて → 金 → 銀 → 銅 → ...

#### ウォッチリスト登録
- **Enter キー**: 銅 → 銀 → 金 → なし → ... （順番に切り替え）
- **1 キー**: 銅に登録
- **2 キー**: 銀に登録
- **3 キー**: 金に登録
- **0 キー**: ウォッチリスト解除

#### 水平線の操作
- **h キー + クリック**: クリック位置に水平線を描画
- **水平線上で右クリック**: 水平線を削除

### マウス操作
- **拡大/縮小**: マウスホイール
- **スクロール**: ドラッグ
- **価格確認**: チャート上をマウスオーバー

### データ更新
毎朝6時（JST）に自動的に最新データに更新されます。
