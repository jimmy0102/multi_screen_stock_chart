# データインポート手順

## 1. Service Keyの取得

Supabaseダッシュボードから：
1. Settings → API
2. 「Service role key」をコピー（`service_role`と書かれた長い文字列）
3. `.env`ファイルに追加：
```
SUPABASE_SERVICE_KEY=あなたのservice_role_key
```

## 2. データインポート実行

```bash
npm run import-data
```

これにより：
- 10個の主要日本株銘柄のマスターデータ
- 各銘柄の過去1年分の日次株価データ

がインポートされます。
